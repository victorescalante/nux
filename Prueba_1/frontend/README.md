# Project

## Create .env

MOCK_API: Is a mock test used for test

```yaml 
MOCK_API=http://jsonplaceholder.typicode.com/
```

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
